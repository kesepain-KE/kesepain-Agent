实现以下要求：
1.运行文件，输入：
    /文件读取 <工作区相对文件地址>
2.输出：
    <文件内容>
3.示例：
    /文件读取 system\plugins\file_read\plugin.json

    {
        "num_1": {
            "command": "/文件读取",
            "explain": "读取工作区内文件内容",
            "run_file": "file_read.py"
        }
    }

<注意>
1.内容搜索限制在此工作区
2.不能读取目录，只能读取文件
3.默认返回文本内容
