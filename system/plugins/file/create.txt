实现以下要求：
[文件查找模式]
    1.运行文件，输入：
        /文件查找 <具体的文件名，支持文件夹>
    2.输出：
        <匹配到的文件或文件夹列表>
    3.示例：
        /文件查找 task

        找到以下匹配项：
        [文件] system/plugins/task/use.txt
        [文件夹] system/plugins/task
    4.注意：
        1.兼容旧指令：/文件搜索
        2.按名称模糊匹配工作区内文件和文件夹
[文件夹内部框架查看模式]
    1.运行文件，输入：
        /文件夹内部框架查看 <具体的文件名>
    2.输出：
        <目录结构>
    3.示例：
        /文件夹内部框架查看 system/plugins/task

        查询结果:
        目录:system/plugins/task
        [目录] run
            [文件] task.py
        [文件] command.json
        [文件] create.txt
        [文件] use.txt
    4.注意：
        1.兼容旧指令：/框架查询
        2.输入文件路径时，会查看该文件所在目录的结构
        3.输入名称而非路径时，会先在工作区查找匹配项
[文件读取模式]
    1.运行文件，输入：
        /文件读取 <相对路径>
    2.输出：
        <文件内容>
    3.示例：
        /文件读取 system/plugins/task/use.txt

        读取结果:
        <文件读取的内容>
    4.注意：
        1.只支持读取工作区内相对路径文件
        2.目录不能直接读取
        3.超长内容会被截断输出
